Discord Electron for Older Systems

A simple, efficient Discord wrapper built using Electron for legacy Windows systems.

Why This Exists

Modern Discord clients no longer support Windows 7 or 8/8.1. This project gives you a working, standalone app that uses the web version of Discord via Electron, designed to work on those older platforms.
